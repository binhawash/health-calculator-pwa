# Health Calculator PWA (Binhawash)

This package is tailored for https://binhawash.github.io/health-calculator-pwa/

## Deploy on GitHub Pages
1) Upload **all files** in this folder (including `assets/`) into repo: `health-calculator-pwa`
2) In repo **Settings → Pages**: Source = Deploy from a branch, Branch = `main`, Folder = `/ (root)`
3) Open: https://binhawash.github.io/health-calculator-pwa/
4) In Safari → Share → Add to Home Screen (iOS)
